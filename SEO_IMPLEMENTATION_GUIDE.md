# 🚀 מדריך SEO מקיף - OR Productions

## 🎯 מה בוצע - שדרוג SEO מלא

### ✅ שדרוגים שהושלמו:

#### 1. 🏷️ **Meta Tags מתקדמים**

- **Title Tags** ממוטבים עם מילות מפתח
- **Meta Descriptions** אטרקטיביים עם CTA
- **Keywords** רלוונטיים לכל עמוד
- **Open Graph** לשיתוף ברשתות חברתיות
- **Twitter Cards** לשיתוף בטוויטר

#### 2. 🗺️ **Sitemap ו-Robots**

- **XML Sitemap** אוטומטי לכל העמודים
- **Robots.txt** מותאם לקידום
- **Priority** ו-**Change Frequency** לכל עמוד

#### 3. 📋 **Structured Data (Schema.org)**

- **Business Schema** - פרטי העסק
- **Product Schema** - מוצרים ומחירים
- **FAQ Schema** - שאלות נפוצות
- **Breadcrumb Schema** - נתיב ניווט
- **Service Schema** - שירותים

#### 4. 🎨 **Meta Tags לכל עמוד**

- **עמוד הבית** - ממוטב למילות מפתח ראשיות
- **עמודי מוצרים** - Title ייחודי לכל מוצר
- **עמודי קטגוריות** - תיאור מותאם לקטגוריה
- **עמודי בלוג** - Article Schema ו-meta tags

## 🛠️ מה צריך לעדכן (פעם אחת)

### 1. **עדכון פרטי העסק:**

```javascript
// components/seo/StructuredData.jsx - שורה 15
"telephone": "+972-XX-XXXXXXX", // ← עדכן למספר הטלפון שלך
"email": "info@orproductions.co.il", // ← עדכן לאימייל שלך
```

### 2. **עדכון כתובת האתר:**

```javascript
// app/sitemap.js - שורה 4
const baseUrl = "https://orproductions.co.il"; // ← עדכן לדומיין שלך
```

### 3. **הוספת קודי אימות:**

```javascript
// app/layout.js - שורות 42-43
"google-site-verification": "YOUR_GOOGLE_VERIFICATION_CODE", // ← מגוגל
"msvalidate.01": "YOUR_BING_VERIFICATION_CODE", // ← מבינג
```

### 4. **עדכון תמונות Open Graph:**

```javascript
// app/layout.js - שורה 22
url: "https://res.cloudinary.com/your-cloud/image/upload/v1/og-image.jpg",
// ← צור תמונה 1200x630 לשיתוף ברשתות
```
